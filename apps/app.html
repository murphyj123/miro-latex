How to use LaTeX for Miro

Editor

Click Editor to open the full LaTeX editor with formatting tools, symbols, and templates.

Convert

Type LaTeX in a text item on the board, select it, then click Convert to render it as an image.

Edit

Select a previously placed LaTeX image, click Edit to load it back into the editor.

Formula Library

Browse or search standard formulae below. Click any formula to open it in the editor.
